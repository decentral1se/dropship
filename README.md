## what is what:

`dropship.py`, run this with python3

`dropship.glade`, UI file, edit with glade

`dropship.css`, additional styling for UI.

## install:

`sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-3.0`

`pip install -r requirments.txt`

## run:

`python3 dropship.py`
